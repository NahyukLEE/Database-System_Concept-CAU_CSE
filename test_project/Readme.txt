20194538 이나혁 (소프트웨어학부 2학년)

Windows OS, JDK 8u261, MySQL 8.0.21 환경입니다.
Gradle을 통해 Java 프로젝트를 관리합니다.
DB접속정보 파일은 auth/mysql.auth 입니다. 
DB접속정보 클래스는 DatabaseAuthInformation.java에 위치합니다.

IDE는 IntelliJ IDEA Ultimate 2020.2.3을 사용하였습니다.

Insert 작업을 수행하는 InsertQueryExample 클래스는 InsertQueryExample.java에 위치하며,
Select 작업을 수행하는 SelectQueryExample 클래스는 SelectQueryExample.java에 위치하고 있습니다.

auth/mysql.auth는 dbdesign_lecture_db 스키마를 위한 DB 접속정보 파일입니다.
